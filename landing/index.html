Security

Built for networks where security is non-negotiable

Manufacturing floors, financial institutions, government agencies — places where cloud AI is not an option.

✓

Complete Air-Gap Support

No external API calls. LLM inference, embeddings, and vector search all run on-premise.

✓

AST-Based SQL Validation

SQL injection prevented at the parse tree level — not bypassable with comments or whitespace tricks.

✓

JWT Authentication

Token-based auth with configurable expiration. No session data stored on client.

✓

Read-Only Database Access

Text-to-SQL only executes SELECT queries. Multi-statement injection blocked at the AST level.
